## 特性变更

### 2.4版本

### 2.3版本

本版本新增或改动了以下功能：

1. 总公司可以查看分公司发票信息，并且可以按照分公司名称进行筛选查看。

2. 发送给受票方的短信通知由两条缩减为一条。

3. 移动开票帮助文档支持手机H5格式。

4. 支持中国人寿客户用“保单号+证件号／金额”的方式查询发票。

###2.3版本
